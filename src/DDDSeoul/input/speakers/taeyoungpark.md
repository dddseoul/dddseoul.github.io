## 🔎 세션 정보

- **세션명**: 스크랩해서 쌓아만 두는 당신에겐 AI Agent가 필요합니다!
- **세션종류**: 브레이크아웃 세션
- **세션설명**:

  웹 서핑을 하다 발견한 좋은 글. 스크랩해서 쌓아만 두는 당신에겐 AI Agent가 필요합니다. 문서를 사람이 직접 분류하고, 요약하며, 때로는 직접 갱신도 하는 시대는 지나갔습니다. 이 발표에서는 Microsoft Agent Framework를 기반으로 대량의 텍스트 자료를 군집화, 분류하는 기술을 소개합니다. 아울러 Agent가 중요한 문서를 선택하고, 검증하며, 새로운 지식으로 확장하는 방법도 소개합니다. 이 발표를 통해, Agent 기반의 자율적인 지식관리 방법을 소개하고, 제가 경험한 시행착오와 인사이트를 나누고자 합니다.

- **세션일정**: 12월 5일(금), 16:00 - 16:40
- **세션장소**: ROOM 2 (동해/독도)
