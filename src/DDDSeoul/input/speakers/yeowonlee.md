## 🔎 세션 정보

- **세션명**: Python in Excel: 데이터 분석의 새로운 시작
- **세션종류**: 브레이크아웃 세션
- **세션설명**:

  Python in Excel 기능을 활용하여 데이터 분석과 자동화를 보다 효율적으로 수행하는 방법을 소개합니다.

  전통적으로 Excel은 접근성이 뛰어나지만 복잡한 데이터 처리에는 한계가 있었습니다. 그러나 Python이 통합되면서 Excel 내에서 직접 데이터 분석, 시각화, 머신러닝 등의 작업을 수행할 수 있게 되었습니다.

  본 세션에서는 Python in Excel의 기본 개념과 주요 기능을 살펴보고 실제 예시를 통해 엑셀과 파이썬의 시너지가 만들어내는 새로운 데이터 분석 환경을 이해하는 시간을 가집니다!

- **세션일정**: 12월 5일(금), 13:30 - 14:10
- **세션장소**: ROOM 1 (제주)
